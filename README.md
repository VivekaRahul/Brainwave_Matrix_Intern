This project is a web-based Daily Planner Website. It's designed to help users organize their daily tasks, schedules, and personal well-being.

It features a user-friendly interface with sections for a TO-DO LIST, a timed SCHEDULE, and a PRIORITY LIST to manage tasks. Additional modules include areas for tracking "MY MOODS", jotting down NOTES, and planning specific work-related items like MEETINGS, PROJECTS, WORK PLAN, and DEADLINES. An APPOINTMENTS section is also available, and the layout is overlaid on a subtle calendar background image. The planner includes interactive elements such as highlighting the current day and allowing users to add new list items dynamically. Crucially, all user-entered information is persistently stored directly within the user's browser using localStorage, ensuring that data remains intact even after closing and reopening the page, thereby eliminating the need for a backend server. The application further enhances usability through features like dynamic list item creation, visual day highlighting, and automatic saving of changes as the user interacts, providing a seamless and reliable personal organization tool.
